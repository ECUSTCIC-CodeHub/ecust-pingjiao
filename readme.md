# ECUST 研究生自动评教助手 - 脚本介绍

这是一个专为华东理工大学研究生设计的油猴脚本，能够自动完成课程教学质量评价，简化评教流程。

## 功能特点

- **自动评分**：自动将所有评分下拉框选为最高分
- **一键操作**：通过悬浮按钮轻松启动自动评教
- **智能提交**：自动查找并点击提交按钮
- **安全可靠**：仅在指定评教页面运行

## 使用方法

1. 安装脚本后访问华东理工大学研究生评教页面
2. 页面右上角会出现绿色"自动评教"悬浮按钮
3. 点击按钮即可自动完成评分并提交

## 技术说明

- 脚本仅在 `graduate.ecust.edu.cn/PostGraduate/WitMis_CourseJxzlpjAddXF.aspx` 页面运行
- 自动识别所有评分下拉框并选择最高分选项
- 自动查找并点击提交按钮完成评教
